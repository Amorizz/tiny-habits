# Tiny Habits — Finance Tracker for Students

A visual, mobile-first finance tracker built for college students. Track spending, set budgets, define savings goals, and build better financial habits — all from a customizable drag-and-drop dashboard.

Built during our exchange semester at **Shanghai University (Spring 2025)** as part of the Innovation Period program.

## Awards

- **Best Public Speaking**
- **Outstanding Business Potential**

## Features

- **Customizable Dashboard** — drag-and-drop widgets (balance, expenses chart, categories breakdown, goals, coffee counter, good decisions tracker). Toggle visibility and resize.
- **Transaction Management** — add income/expenses with category, date, need/want classification, and notes. Filter and search through history.
- **Budget Tracking** — set monthly budgets per category with progress bars and spending alerts.
- **Savings Goals** — define financial goals with target amounts, deadlines, and visual progress.
- **Category System** — default categories (Food, Transport, Entertainment, Shopping, Health, Income) with AI-powered auto-categorization via OpenAI.
- **Smart Widgets** — coffee count tracker, good financial decisions counter, spending tips, and challenge system to gamify saving.

## Tech Stack

- **Next.js 15** + **React 19** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui** — styling and component library
- **Recharts** — expense charts and visualizations
- **@hello-pangea/dnd** — drag-and-drop dashboard customization
- **OpenAI API** — transaction auto-categorization
- **Zod** + **React Hook Form** — form validation
