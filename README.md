
# Tiny Habits: Finance Tracking App

Tiny Habits is a simple and visually appealing finance tracker designed for college students. Track your spending, set goals, and manage your budget with ease.

This MVP was created during our exchange semester at Shanghai University (Spring 2025) as part of the Innovation Period.

## Running the Project Locally

To run this project locally, follow these steps:

1. **Install dependencies** (using [pnpm](https://pnpm.io/) is recommended):
   ```sh
   pnpm install
   ```
   Or with npm:
   ```sh
   npm install
   ```
   Or with yarn:
   ```sh
   yarn install
   ```

2. **Start the development server:**
   ```sh
   pnpm dev
   ```
   Or with npm:
   ```sh
   npm run dev
   ```
   Or with yarn:
   ```sh
   yarn dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

4. To build for production:
   ```sh
   pnpm build
   pnpm start
   ```
   (Or use `npm run build && npm start` or `yarn build && yarn start`)
